Printed Text Scanner

This project is a simple GUI application that scans printed text using PyTesseract. It allows you to load images or use a live camera, select a region of interest, run OCR, and view the extracted text directly inside the interface. The app also provides an overlay preview so you can see exactly what text was detected.

Features

GUI interface

Image loading

Live camera input

OCR text extraction

ROI (region of interest) selection

Overlay preview of detected text

Easy-to-read text display inside the app

Purpose

This project was created as part of an AI Without ML assignment to explore and understand how classical OCR systems work.

How It Works

Open the app, load an image or use your camera, select the area you want scanned, and let the OCR engine extract the text for you. The results appear instantly in the GUI.
